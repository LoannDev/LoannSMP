Support & Aide
Si ton launcher ne se lance pas, essaie de suivre ses étapes pour voir si ça fonctionne ou si t'as la flemme passe direct à l'étape 3

1. Assure-toi d'avoir installé la dernière version de Java (Java 17+ de préférence). Tu peux la télécharger ici : Télécharger Java (je n'ai pas essayé mais c'est peut-être ça)

2. Assure-toi que ton antivirus ne bloque pas le launcher. Parfois, les antivirus peuvent détecter à tort le launcher comme une menace, car un fichier Python compilé en .exe est souvent flag, même si c'est pas toujours le cas. Essaie de désactiver temporairement ton antivirus ou d'ajouter une exception pour le launcher.

3. Si l'erreur persiste, utilise la méthode Installation par Script disponible sur la page d'accueil. Elle affichera une console qui nous permettra de diagnostiquer l'erreur précise.
